# F S F APP

Helped the federation to manage the match score

## Running in local

#### Prerequisites

Install Node.js 4 +

Mongo DB 3.6 +

#### Clone project 

git clone https://github.com/VihoSerge/fsf-app.git

#### Install package

Go to the project folder and tap

npm install

#### Start project

node server/server.js

npm start 

#### Open browser

http://localhost:4200/dashboard
